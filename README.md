# 📌 Sistema de Gestión de Clientes (C++)

Proyecto desarrollado en **C++** como práctica de programación estructurada, manejo de archivos binarios y gestión de datos en memoria.

---

## 🧩 Funcionalidades principales
- ➕ **Alta de clientes**
- 📋 **Listado completo**
- 🔍 **Búsqueda por ID**
- 🔄 **Ordenamiento por ID**
- 📊 **Estadísticas**
  - Total de clientes
  - Clientes activos
  - Promedio de edad
- 💾 **Guardado en archivo binario (`productos.dat`)**
- 📂 **Carga automática desde archivo**
- 🚪 **Salida segura**

---

## 🛠️ Tecnologías utilizadas
- Lenguaje: **C++**
- Manejo de archivos binarios
- Estructuras (`struct`)
- Funciones y modularización
- Ordenamiento y búsqueda secuencial

---

## ▶️ Cómo compilar
```bash
g++ gestionClientes.cpp -o gestionClientes.exe
```

## ▶️ Cómo ejecutar
```bash
gestionClientes.exe
```

---

## 📁 Archivos incluidos
- `gestionClientes.cpp` — Código fuente principal  
- `productos.dat` — Archivo binario generado automáticamente
